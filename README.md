### Instructions to Start Airflow

1. **Start the Airflow environment**:  
   Run the following command in your terminal:  
   ```bash
   docker-compose up
    ```
2. **Access the Airflow web interface:**
   Open your web browser and navigate to: http://localhost:8080/
3. **Log in to Airflow:**
   - Username: airflow
   - Password: airflow
